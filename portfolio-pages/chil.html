orion spacecraft AR object locator - LM CHIL



description


Project completed in partnership with NASA/Orion and Lockheed Martin's CHIL (Collaborative Human Immersive Lab). The augumented reality (AR) application allows 'astronauts' aboard Orion to more easily identify the storage container for various items the crew may need. Future work can involve updating/recommending new item storage locations as items are used to preserve healthy cabin weight distribution.

skills used/developed


- Unity/C#
- Augmented Reality
- Microsoft HoloLens
- Human Factors Engineering

documentation


This project was completed independently with support from the CHIL lab. Unity code was written to read an external CSV containing container/sub-container/item names. Editing this csv would update the item "Main Menu" the user sees in AR attached to their wrist in real time, allowing for easy storage location changes. The user can navigate through the menu to select the desired item. The item storage location(s) will then change color, indicating where the item is located. The Unity models in the following demo have been simplified to boxes to protect proprietary information. The demo was recorded through keyboard controls.
